# Phase Gate

1. Open the relevant control sheet in docs/delivery/.
2. Compare implementation against:
   - phase objective
   - expected deliverable
   - manual check
   - smoke test
3. Report pass/fail for:
   - scope gate
   - usability gate
   - stability gate
   - regression gate
   - smoke gate
4. If any fail, do not proceed to next phase.
5. If all pass, summarize readiness for next phase.
