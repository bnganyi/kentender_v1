# Implement Phase

1. Read the nearest AGENTS.md files that apply.
2. Read the relevant module docs:
   - PRD
   - UX rules
   - data model
   - implementation sequence
3. Read the current phase prompt in docs/prompts/.
4. Restate:
   - current phase
   - what is in scope
   - what is out of scope
   - files likely to change
5. Implement only that phase.
6. Stop and summarize:
   - files changed
   - what changed
   - what was intentionally not changed
   - manual checks to run
   - smoke tests to run
