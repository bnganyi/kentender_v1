KenTender UX rules:
- Workspace-first, not raw form-first
- No long scrolling forms for complex flows
- One clear next step at all times
- Use tables for structured data
- Use modal/panel entry for add/edit
- Show parent-child relationships clearly
- Make workflow state visible
- Do not rely on search as the main entry path
